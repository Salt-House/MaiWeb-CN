# MaiWeb-CN
乌蒙大象中文玩家站

## 前端技术栈
- Next.js
- TailwindCSS

## 项目结构

```
.
├── next-env.d.ts 
├── next.config.mjs 
├── package.json             #相关依赖及命令
├── postcss.config.mjs      
├── public                   #存放服务器静态资源
│   ├── img
├── src                      #NextJs源码资源       
│   └── app                  #Next网页应用源码
│       ├── api              #Api目录
│       ├── components       #组件目录
│       ├── globals.css      #全局CSS
│       ├── introduction     #介绍页面目录
│       ├── layout.tsx       #全局布局
│       ├── loading.tsx      #加载组件
│       ├── page.tsx         #主页内容
│       └── questionnaire    问卷目录
├── tailwind.config.ts
└── tsconfig.json
```
